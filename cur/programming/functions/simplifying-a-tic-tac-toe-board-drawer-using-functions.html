Wir haben in der Vergangenheit einen repeat Block verwendet, um die Wiederholung von Code zu vermeiden. Ebenso können wir einen selbstgebauten Block verwenden, um Code-Wiederholung zu vermeiden. Unten ist Code, um ein Tic-Tac-Toe Brett zu zeichnen. Deine Aufgabe ist, Blöcke zu erzeugen, die diesen Code vereinfachen.

Ein paar Sachen zur Erinnerung:

  • Unterteile zuerst die Zeichen-Aufgabe in sinnvolle Unter-Aufgaben. Wir empfehlen, das mit Papier und Bleistift zu erledigen, ohne dabei auf den Code zu schauen!
  • Gib Deinen neuen Blöcken passende Namen. Du möchtest die Lektüre des Codes vereinfachen—am besten so einfach wie natürliche Sprache—und ihn damit verständlich machen.
  • Es könnte Sinn machen, einige Blöcke in einzelne selbstgebaute Blöcke zu zerlegen. Zum Beispiel, könnte ein einzelnes besser durchgeführt werden in Form von in einem selbstgebauten Block, gefolgt von in einem anderen.

Verwende dieses snap Projekt, ein Duplikat des obigen Bilds, als Ausgangspunkt.

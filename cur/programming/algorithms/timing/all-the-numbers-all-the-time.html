In dieser Sektion werden wir dieses Timing-Framework nutzen. Es ähnelt dem, welches du vorhin erarbeitet hast, aber es enthält die Vorlagen für ein paar Blöcke, die wir nun ausfüllen werden. Der erste Block dafür ist bereits auf der Bühne:

Dieser Block sollte die eingegebene Liste mit allen Zahlen von start bis end füllen.  Wenn er richtig funktioniert, sollte der Block auf der Bühne alle Zahlen von 1 bis max ausgeben, wobei max die Zahl ist, die wir verändern werden, wenn wir mit der Laufzeit experimentieren.

    First, talk to your partner and discuss an algorithm that you can use to fill the input list with all of the numbers from start to end.  Once you both have decided on an algorithm, complete the body of the block.  Now, run the script with max set to 10.  Run it a few times to get an idea of the average time the computer takes to generate this list (to the nearest tenth of a second); you don't need to be exact!  Repeat the experiment with max set to 20, 40, 100, 200, and 1000.  Do you see a pattern?

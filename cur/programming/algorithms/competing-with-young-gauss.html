Wir möchten alle Zahlen von 1 bis N summieren, wobei N eine sehr große Zahl sein könnte. Der offensichtlichste Weg wäre, sie einzeln aufzuaddieren.

Wir können aber auch in die Fußstapfen des jungen Johann Carl Friedrich Gauss treten, einem berühmten Mathematiker aus dem 18. Jahrhundert. Er wurde von seinem Grundschullehrer dazu verdammt, alle Zahlen von 1 bis 100 zusammenzuzählen. Der Lehrer, J. G. Büttner, ging davon aus, ihn damit eine lange Zeit zu beschäftigen. Aber Gauss löste die Aufgabe in wenigen Sekunden. Ihm wurde klar, dass 1 + 2 + 3 ... + 100 dasselbe ist wie (1 + 100) + (2 + 99) + (3 + 98) + ... + (50 + 51). Die Summe jedes Zahlenpaars beträgt 101, es gab 50 solcher Paare, also musste 101 × 50 = 5050 die Antwort sein.

In general, if you wanted to add the numbers from 1 to N, the answer would be
     (N + 1) × (N ÷ 2),
because there are (N ÷ 2) pairs, each of which add up to (N + 1).

Imagine that you are playing a game against young Gauss to see who could add numbers from 1 to N faster: he does it his way, while you manually add them one by one. You can either win, lose, or tie in this game. What do you think would happen if N is 2? If N is 10? If N is 100? If N is one million?

Suche in der Variablen-Palette unter den orangen Variablen-Blöcke nach einer Gruppe roter Blöcke. Diese Blöcke betreffen Listen. Mit einer Liste kann man verschiedene Werte zusammenfassen.

Der erste rote Block ist der list Block, mit dem eine Liste gemacht wird:

Ziehe einen list Block in die Skript-Region, um damit zu experimentieren.

Dieser Block verwendet eine snap Eigenschaft, die Du schon im join Block gesehen hast: eine vaiadic Eingabe. Das bedeutet, obwohl Du nur ein Eingabefeld im Block siehst, kannst Du ihm beliebig viele Eingaben geben. Mit die Links- und Rechtspfeilen am Ende des Blocks kann man Eingabefelder hinzufügen oder löschen:



Wie jeder Reporterblock, kann der list Block in das Eingabefeld eines anderen Blocks gezogen werden:

Das graue abgerundete Rechteck mit roten abgerundeten Rechtecken innen, stellt eine Liste dar. Jedes rote Rechteck ist ein Listenelement.

Das Listenbild hat mehrere Extrawidgets: ein + Knopf, ein Pfeil nach unten, und so weiter. Wenn ein Listenwert in einem Variablenbeobachter zu sehen ist, kannst Du mit diesen Kontrollknöpfen den Inhalt der Liste direkt verändern. Aber das machen wir erst einmal noch nicht; also konzentriere Dich vorläufig einfach auf die Werte in den roten Rechtecken.

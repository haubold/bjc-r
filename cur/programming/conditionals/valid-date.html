Der unten gezeigte Code prüft, ob seine Argumente ein korrektes Datum im Gregorianischen Kalender (der den wir in Deutschland verwenden) darstellen. Dieser Kalender wurde 1582 eingeführt, so dass nur Daten ab diesem Jahr gelten. Vereinfache den valid-date Block und mache ihn lesbarer, indem Du eine Hilfprozedur mit einem sinnvollen Namen definierst (Tipp: Ist Schaltjahr?). Wenn Du fertig bist, sollte es mehr Code geben als vorher, aber er sollte leichter zu verstehen sein.

Wichtig: Wie dieses Beispiel zeigt, kann es sehr vorteilhaft sein, Deine eigenen Blöcke zu verwenden. Das Vorgehen in dieser Übung heiß&t passender Weise funktionale Zerlegung. Dabei zerlegen wir größere Funktion in kleinere Stücke, die wir dann wieder zur ursprünglichn Funktion zusammenbauen. Das ist eine Methode der Abstraktion, da die größere Funktion (valid-date?) nicht zu wissen braucht, was die Funktionen aus denen sie besteht tun, sie kann einfach davon ausgehen, dass sie funktionieren.

Hier sind sechs Aufgaben, damit es sich ein bisschen mehr wie ein Mensch verhält. Wenn dir selbst noch etwas einfällt, was das ganze viel cooler macht, bau es doch auch einfach ein. Versuche diese Aufgaben nacheinander zu lösen. Wenn du fertig bist, sollte das Programm alles beinhalten.

Aufgabe #1: Der Sprite sollte den Spieler am Anfang begrüßen und nach dem Namen fragen.

Aufgabe #2: Wenn der Spieler die falsche Zahl eingibt, sollte der Sprite den Spieler beim Namen ansprechen und ihm das sagen. Falls beispielsweise Karl-Heinz das Spiel spielt, könnte die Nachricht Schade Karl-Heinz, das ist leider nicht die richtige Antwort lauten.

Aufgabe #3: Wenn der Spieler die falsche Zahl eingibt, sollte der Sprite sagen, ob die Zahl größer oder kleiner ist.

Challenge #4: Right now, the script always picks a number between 1 and 10. Change this so that it picks a number between 1 and a variable named maximum. Ask the player what they would like the maximum number to be, before choosing a random number. Use this maximum number as the highest number that the script will choose.

Challenge #5: Keep track of how many guesses it takes before the player guesses the right number.

Challenge #6: When the player guesses the secret number, tell them how many guesses it took, and congratulate them using their name.
